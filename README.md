### 2. Workplace Safety Analysis

**Tools:** Excel · CSV analysis · KPI design · risk scoring  
**Repo:** `Work_Safety`

Analyzed 2012 U.S. workplace safety data by state and built a composite risk score using fatality rate, injury rate, inspection delay, and penalty data.

Key findings:

- Highest composite risk score: West Virginia
- Highest fatality rate: North Dakota
- Median state fatality rate: 3.5
- U.S. injuries per fatality: 625.9

**Recruiter signal:** Excel/CSV analysis, metric design, and business-oriented summary reporting.

### 3. Austin Construction Permit Growth Analysis

**Tools:** Python · pandas · scikit-learn · exploratory data analysis · modeling  
**Repo:** `Austin-Growth`

Analyzed Austin construction permit data to identify ZIP codes with concentrated development activity and model added square footage as a proxy for construction intensity.

The project combines:

- permit-level data cleaning
- ZIP-code and geographic summaries
- exploratory analysis
- random forest regression prototype
- cautious interpretation of model performance

**Recruiter signal:** Python EDA and modeling on civic/open-data records.
